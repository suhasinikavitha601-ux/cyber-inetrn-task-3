Phishing Email Detection Model

Description

This project uses Machine Learning to classify emails as either phishing or safe. The model is trained using a dataset of email messages and uses TF-IDF vectorization with Logistic Regression for classification.

Features

- Detects phishing emails
- Classifies safe and malicious emails
- Calculates model accuracy
- Displays a confusion matrix
- Allows testing with custom email messages

Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

Project Structure

Phishing_Email_Detection/

├── phishing_detector.py

├── emails.csv

├── requirements.txt

└── README.md

Installation

Install the required packages:

pip install -r requirements.txt

Run the Project

python phishing_detector.py

Output

- Accuracy Score
- Email Classification Result
- Confusion Matrix Visualization
